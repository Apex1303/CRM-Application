# Simple Java CRM (GUI Edition)

A lightweight Customer Relationship Management (CRM) application built with Java Swing. This project demonstrates basic CRUD (Create, Read, Delete) operations, object-oriented principles, and GUI development in Java.

## ✨ Features

- **Modern GUI**: Uses the system's native Look and Feel for a professional appearance.
- **Customer Management**: Add and delete customers via an intuitive table interface.
- **Validation**: Includes basic input validation and error dialogs.
- **Thread Safety**: Implements `SwingUtilities` for a smooth, non-freezing UI.

## 🛠️ Tech Stack

- **Language**: Java 17
- **Library**: Java Swing (AWT)
- **Build System**: Manual (`javac`) or Maven

## 🚀 Getting Started

### Prerequisites

- Java Development Kit (JDK) 17 or higher installed.

### How to Run (Using `javac`)

1. **Clone the repository:**
```bash
   git clone https://github.com/your-username/simple-crm.git
   cd simple-crm
```

2. **Compile the source files:**
```bash
   javac -d out src/main/java/com/crm/model/Customer.java src/main/java/com/crm/ui/CRMFrame.java src/main/java/com/crm/CRMApp.java
```

3. **Run the application:**
```bash
   java -cp out com.crm.CRMApp
```
